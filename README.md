# 🚀 Open Source Contribution for Beginners

Welcome! 👋

This repository is designed to help students learn the complete workflow of contributing to open source projects on GitHub.

Whether you're making your **first contribution** or helping others get started, this project provides a safe environment to practice the contribution process.

---

## 🎯 Objective

By contributing to this repository, you will learn how to:

* Fork a repository
* Clone a repository locally
* Create a new branch
* Make changes
* Commit your changes
* Push changes to GitHub
* Create a Pull Request (PR)
* Collaborate with maintainers

---

## 📌 First Contribution Task

Add your name and details to the `contributors.md` file.

Example:

```md
## John Doe

- GitHub: @johndoe
- College: XYZ University
- Favorite Technology: Python
```

---

## 🛠 Prerequisites

Before starting, make sure you have:

* A GitHub account
* Git installed on your computer
* Basic knowledge of Git commands

### Install Git

* Windows: https://git-scm.com/download/win
* macOS: https://git-scm.com/download/mac
* Linux: https://git-scm.com/download/linux

---

## 🍴 Step 1: Fork the Repository

Click the **Fork** button in the top-right corner of this repository.

This creates your own copy of the project.

---

## 📥 Step 2: Clone Your Fork

Replace `<your-username>` with your GitHub username.

```bash
git clone https://github.com/<your-username>/<repository-name>.git
```

Move into the project folder:

```bash
cd <repository-name>
```

---

## 🌿 Step 3: Create a New Branch

Create a branch for your contribution.

```bash
git checkout -b add-your-name
```

Example:

```bash
git checkout -b add-john-doe
```

---

## ✏️ Step 4: Make Your Changes

Open the `contributors.md` file.

Add your details following the existing format.

Save the file.

---

## ✅ Step 5: Commit Your Changes

Check modified files:

```bash
git status
```

Stage your changes:

```bash
git add .
```

Commit:

```bash
git commit -m "Add John Doe to contributors list"
```

---

## 🚀 Step 6: Push Your Branch

```bash
git push origin add-your-name
```

Example:

```bash
git push origin add-john-doe
```

---

## 🔄 Step 7: Create a Pull Request

1. Go to your fork on GitHub.
2. Click **Compare & Pull Request**.
3. Add a meaningful title.
4. Submit the Pull Request.

Example title:

```text
Add John Doe to contributors list
```

---

## 📋 Contribution Guidelines

* Make only the requested changes.
* Do not remove other contributors' information.
* Use meaningful commit messages.
* Follow the repository structure.
* Be respectful and collaborative.

---

## 🧑‍💻 Git Commands Cheat Sheet

```bash
# Clone repository
git clone <repo-url>

# Create branch
git checkout -b branch-name

# Check status
git status

# Stage changes
git add .

# Commit changes
git commit -m "your message"

# Push changes
git push origin branch-name
```
